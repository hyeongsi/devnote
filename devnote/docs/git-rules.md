# Git Rules

## Commit Convention

- feat: 기능 추가
- fix: 버그 수정
- refactor: 리팩토링
- docs: 문서 수정
- style: 스타일 변경
- chore: 기타 설정
- test: 테스트

## Commit Message Rules

- 모든 커밋 메시지는 한글로 작성
- 한 줄 요약은 명확하게 작성
- 불필요한 영어 혼합 금지

## Pull Request Rules

- PR 제목은 한글 작성
- `[codex]` 사용 금지
- PR 본문 한글 작성
- 작업 내용 / 변경 사항 / 테스트 여부 포함
